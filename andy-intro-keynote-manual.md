## 🛠️ 创建方法

### 方法 1：手动创建（推荐）

1. **打开 Keynote**
2. **选择主题** - 选择简洁的基础主题
3. **创建 6 张幻灯片** - 按上述内容填写
4. **保存** - 保存为 `Andy-Intro.key`

### 方法 2：使用 Python 脚本（需要 PyAutoGUI）

**前提条件：**
- macOS 10.15+
- Python 3.9+
- Keynote 已安装

**安装：**
```bash
# 使用 Homebrew 安装 Python 3.9+
brew install python@3.11

# 安装依赖
pip3 install pyautogui pillow pyscreeze
```

**授予权限：**
1. 系统偏好设置 → 安全性与隐私 → 隐私
2. 辅助功能 → 添加 Terminal

**运行：**
```bash
python3 andy-intro-keynote-automation.py
```

### 方法 3：导出为 PDF

创建完成后可导出为 PDF 分享：
1. 文件 → 导出到 → PDF
2. 选择质量
3. 保存

## 📝 设计建议

- **配色：** 保持专业，使用蓝色/灰色系
